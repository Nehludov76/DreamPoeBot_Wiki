# NotificationInfo Properties

The NotificationHudElement NotificationInfo type exposes the following members.


|  | Name | Description |
| --- | --- | --- |
|  | AcceptPos | The position of the accept button, in case you want to handle it yourself. |
|  | AccountName | Account name of the sender. |
|  | CharArea | The actual Area of the sender. |
|  | CharLevel | Char level of the sender. |
|  | CharName | Char name of the sender. |
|  | DeclinePos | The position of the decline button, in case you want to handle it yourself. |
|  | GuildName | Guild name of the sender. |
|  | IsVisible | True if the popup is fully visible. It is highly advice to sleep for 700-1000ms after this become true, to ensure the popup has stopped moving. |
|  | NotificationTypeEnum | Enumerator representing the Trade type. |
|  | Type | String representing the notification Type. |
