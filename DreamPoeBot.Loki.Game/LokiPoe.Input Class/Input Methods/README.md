# Input Methods

The LokiPoe Input type exposes the following members.


|  | Name | Description |
| --- | --- | --- |
|  | ClickLMB | Click the left mouse button, Pls NOTE: Its important to pass the X and Y location if you click on a User Interface control, while is not necessary if you click on the game world. |
|  | ClickMMB | Click the middle mouse button, Pls NOTE: Its important to pass the X and Y location if you click on a User Interface control, while is not necessary if you click on the game world. |
|  | ClickRMB | Click the right mouse button, Pls NOTE: Its important to pass the X and Y location if you click on a User Interface control, while is not necessary if you click on the game world. |
|  | DblClickLMB | Double Click the left mouse button, Pls NOTE: Its important to pass the X and Y location if you click on a User Interface control, while is not necessary if you click on the game world. |
|  | DblClickMMB | Double Click the middle mouse button, Pls NOTE: Its important to pass the X and Y location if you click on a User Interface control, while is not necessary if you click on the game world. |
|  | DblClickRMB | Double Click the right mouse button, Pls NOTE: Its important to pass the X and Y location if you click on a User Interface control, while is not necessary if you click on the game world. |
|  | Equals | Determines whether the specified object is equal to the current object. (Inherited from Object .) |
|  | Finalize | Allows an object to try to free resources and perform other cleanup operations before it is reclaimed by garbage collection. (Inherited from Object .) |
|  | GetClickableHighlightLabelDimensions | Return the position and dimension of a NetworkObject label, or Vector2.Zero if the lebel is not visible. |
|  | GetClickableHighlightLabelPosition | Return the position of a NetworkObject label, or Vector2.Zero if the lebel is not visible. |
|  | GetHashCode | Serves as the default hash function. (Inherited from Object .) |
|  | GetKeyForAction | Retrieve the currently configured key for a specifig game action. the string passed must be in the same exact form as show in the game options. Ex: "Atlas Screen", "Pantheon Panel", "Attack in place". PLEASE NOTE: even if this is supposed to work even with customized key setup, I strongly sugest you to keep the DEFAULT key setup to ensure the bot work. |
|  | GetType | Gets the Type of the current instance. (Inherited from Object .) |
|  | HasVisibleHighlightLabel | Return true if the NetworkObject has a visible label ingame. |
|  | HighlightObject | This function will move the mouse over a specific NetworkObject. This is a key function and must be used to loot, attack or interact with the game world. |
|  | MemberwiseClone | Creates a shallow copy of the current Object . (Inherited from Object .) |
|  | MoveMouse(Int32, Int32) | Obsolete. Not used anymore, use MouseManager class to handle all mouse movments. |
|  | MoveMouse(Single, Single) | Obsolete. Not used anymore, use MouseManager class to handle all mouse movments. |
|  | MoveMouseToScreenPos(Vector2) | Move the mouse to a specific location using screen position, this method is rarely used. |
|  | MoveMouseToScreenPos(Vector2i) | Move the mouse to a specific location using screen position. |
|  | MoveMouseToScreenPos(Int32, Int32, Boolean) | Move the mouse to a specific location using screen position. |
|  | MoveMouseToWorldPos(String, Vector2) | Move the mouse to a specific location using world position (position of game objects) This method take care of converting the world position accounting for the current camera zoom and rotation, to screen position. |
|  | MoveMouseToWorldPos(String, Vector2i) | Move the mouse to a specific location using world position (position of game objects) This method take care of converting the world position accounting for the current camera zoom and rotation, to screen position. |
|  | MoveMouseToWorldPos(String, Vector3) | Move the mouse to a specific location using world position (position of game objects) This method take care of converting the world position accounting for the current camera zoom and rotation, to screen position. |
|  | PasteText | Paste a specific text into a text field. |
|  | ResetKeysState | Reset all the keys to unpressed. |
|  | RunInSta |  |
|  | SimulateKeyEvent(Char) | Press a key using the char rapresentation of that key. this is usually used to send specific keystroke, like `a`, `b` `c` etc etc. |
|  | SimulateKeyEvent(Keys, Boolean, Boolean, Boolean, Keys) | Press a key using System.Window.Form.Keys as parameter, this is used to send Keystroke for special keys like `Enter`, `Esc` etc. The most common usage for this method is `SimulateKeyEvent(Keys.Escape, true, false, false); that simulate a single keystroke.` |
|  | ToString | Returns a string that represents the current object. (Inherited from Object .) |
