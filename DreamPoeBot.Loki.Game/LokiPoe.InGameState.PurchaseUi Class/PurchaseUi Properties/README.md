# PurchaseUi Properties

The LokiPoe InGameState PurchaseUi type exposes the following members.


| Name | Description |
| --- | --- |
| InventoryControl | Return a InventoryControl that let you interact with the current Tab content. |
| IsOpened | Return true if the panel is open. |
| TabControl | Return a TabControlWrapper that describe the Tab controll of the panel. |
