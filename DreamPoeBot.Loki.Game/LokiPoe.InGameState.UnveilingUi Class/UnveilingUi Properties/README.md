# UnveilingUi Properties

The LokiPoe InGameState UnveilingUi type exposes the following members.


| Name | Description |
| --- | --- |
| AvailableMods | Return a List of DatModsWrapper that describe the available Mods. |
| InventoryControl_Main | Return a InventoryControlWrapper to interact with the Item slot. |
| IsConfirmUnveilButtonEnabled | Return true if the Confirm button is visible AND enabled. |
| IsConfirmUnveilButtonVisible | Return true if the Confirm button is visible. |
| IsOpened | Return true if the interface is open. |
| IsUnveilButtonEnabled | Return true if the Unveil button is visible AND enabled. |
| IsUnveilButtonVisible | Return true if the Unveil button is visible. |
| SelectedMods | Return a DatModsWrapper that describe the actually selected Mods, or null if none are selected. |
