# SyndicateUI Methods

The LokiPoe InGameState SyndicateUI type exposes the following members.


| Name | Description |
| --- | --- |
| Bargain | Select the Bargain option. |
| Betray | Select the Betray option. |
| Execute | Select the Execute option. |
| Interrogate | Select the Interrogate option. |
| Release | Select the Release option. |
| SafeHouse | Click the Safehouse button to summon the safehouse portal. |
