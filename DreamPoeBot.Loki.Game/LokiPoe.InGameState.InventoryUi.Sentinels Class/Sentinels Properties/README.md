# Sentinels Properties

The LokiPoe InGameState InventoryUi Sentinels type exposes the following members.


|  | Name | Description |
| --- | --- | --- |
|  | InventoryControl_ApexSentinel | Return a InventoryControlWrapper describing your ApexSentinel inventory. |
|  | InventoryControl_PandemoniumSentinel | Return a InventoryControlWrapper describing your PandemoniumSentinel inventory. |
|  | InventoryControl_StalkerSentinel | Return a InventoryControlWrapper describing your StalkerSentinel inventory. |
|  | IsApexSentinelSlotVisible | Return true if the ApexSentinel slot is Visible and you can slot a sentinel in it. |
|  | IsPandemoniumSentinelSlotVisible | Return true if the PandemoniumSentinel slot is Visible and you can slot a sentinel in it. |
|  | IsStalkerSentinelSlotVisible | Return true if the StalkerSentinel slot is Visible and you can slot a sentinel in it. |
