# Beast Properties

| --- | --- | --- |

|  | IsVisible | Return true if this beast is visible (not filtered by the search text). Please note, The beast is visible even if not actually in the visible portion of the scrollbox. be sure to always use strict search text. |
|  | Level | The level of this beast. |
