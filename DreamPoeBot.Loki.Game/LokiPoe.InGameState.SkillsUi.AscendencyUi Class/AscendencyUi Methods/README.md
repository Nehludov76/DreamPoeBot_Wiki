# AscendencyUi Methods

The LokiPoe InGameState SkillsUi AscendencyUi type exposes the following members.


| Name | Description |
| --- | --- |
| ChooseAscendancyPassive | This function let you allocate an ascendancy point. |
| RefundAscendancyPassive | Click on a specific ascendancy point to deallocate it. Remember to EnableRefound() before to call this, and to ConfirmOperation() once done. |
| Toggle | Click the button that open or close the Ascendancy passive ui. |
