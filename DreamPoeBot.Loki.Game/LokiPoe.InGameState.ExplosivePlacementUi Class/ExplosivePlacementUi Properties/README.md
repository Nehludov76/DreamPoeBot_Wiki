# ExplosivePlacementUi Properties

The LokiPoe InGameState ExplosivePlacementUi type exposes the following members.


| Name | Description |
| --- | --- |
| ExplosiveRemaining | Return the numbers of explosive remaining. |
| IsEnabled | Return true if the placement button is enabled. |
| IsOpened | Return true if this panel is visible, Please note, this panel is almost always visible, you should check IsEnabled to know if the TogglePlacement button is actually interactable. |
| IsPlacementActive | Return true if the placement is active and the cursor has the placement marker. |
| IsUndoAvaiable | Return true if the undo button is enabled. |
