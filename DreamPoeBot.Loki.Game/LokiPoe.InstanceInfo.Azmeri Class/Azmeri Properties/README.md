# Azmeri Properties

The LokiPoe InstanceInfo Azmeri type exposes the following members.


|  | Name | Description |
| --- | --- | --- |
|  | CurrentFuel | Return A integer, representing the current glowing circle radius in Azmeri wood, This number is not directly the Current `Fuel`, but can be associated 1 o 1 to that value and so used for the purpose. |
|  | CurrentPrimalWisps | Return a integer, representing the actual Primal Wisp collected. |
|  | CurrentVividWisps | Return a integer, representing the actual Vivid Wisp collected. |
|  | CurrentWildWisps | Return a integer, representing the actual Wild Wisp collected. |
|  | FuelPct | Return a double, with 2 digit precision, represent the current Pct of fuel left. |
|  | MaxFuel | Return A integer, representing the max glowing circle radius in Azmeri wood, This number is not directly the Max `Fuel`, but can be associated 1 o 1 to that value and so used for the purpose. |
