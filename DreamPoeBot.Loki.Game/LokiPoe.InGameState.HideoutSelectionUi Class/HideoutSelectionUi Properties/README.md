# HideoutSelectionUi Properties

The LokiPoe InGameState HideoutSelectionUi type exposes the following members.


| Name | Description |
| --- | --- |
| AllHideouts | Return a IEnumerable of HideoutTemplate describing all the templates (locked and unlocked). |
| IsOpened | Return true when the Hideout Selection panel is displayed. |
| UnlockedHideouts | Return a IEnumerable of HideoutTemplate describing all the already UnlockedHideouts templates. |
