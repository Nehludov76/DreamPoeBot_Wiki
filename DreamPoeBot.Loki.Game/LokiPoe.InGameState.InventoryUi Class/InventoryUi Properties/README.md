# InventoryUi Properties

The LokiPoe InGameState InventoryUi type exposes the following members.


|  | Name | Description |
| --- | --- | --- |
|  | AllInventoryControls | Return a IEnumerable of InventoryControlWrapper listing ALL your inventory (the main inventory and all equipped slots) |
|  | AreIconsDisplayed | This info is relative to the Gems level up shoved at the bottom of the inventory panel Return true if at least 1 gem is displayed |
|  | InventoryControl_Belt | Return a InventoryControlWrapper describing your Belt inventory. |
|  | InventoryControl_Boots | Return a InventoryControlWrapper describing your Boots inventory. |
|  | InventoryControl_Chest | Return a InventoryControlWrapper describing your Chest inventory. |
|  | InventoryControl_Flasks | Return a InventoryControlWrapper describing your Flasks inventory. |
|  | InventoryControl_Gloves | Return a InventoryControlWrapper describing your Gloves inventory. |
|  | InventoryControl_Graft | Return a InventoryControlWrapper describing your trinket inventory. |
|  | InventoryControl_Head | Return a InventoryControlWrapper describing your Head inventory. |
|  | InventoryControl_LeftRing | Return a InventoryControlWrapper describing your LeftRing inventory. |
|  | InventoryControl_Main | Return a InventoryControlWrapper describing your main inventory. |
|  | InventoryControl_Neck | Return a InventoryControlWrapper describing your Neck inventory. |
|  | InventoryControl_PrimaryMainHand | Return a InventoryControlWrapper describing your PrimaryMainHand inventory. |
|  | InventoryControl_PrimaryOffHand | Return a InventoryControlWrapper describing your PrimaryOffHand inventory. |
|  | InventoryControl_RightRing | Return a InventoryControlWrapper describing your RightRing inventory. |
|  | InventoryControl_SecondaryMainHand | Return a InventoryControlWrapper describing your SecondaryMainHand inventory. |
|  | InventoryControl_SecondaryOffHand | Return a InventoryControlWrapper describing your SecondaryOffHand inventory. |
|  | InventoryControl_TherdRing | Return a InventoryControlWrapper describing your TherdRing inventory. |
|  | InventoryControl_Trinket | Return a InventoryControlWrapper describing your trinket inventory. |
|  | IsOpened | Return true if the inventory panel is open. |
