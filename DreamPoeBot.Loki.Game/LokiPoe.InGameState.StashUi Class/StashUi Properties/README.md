# StashUi Properties

The LokiPoe InGameState StashUi type exposes the following members.


|  | Name | Description |
| --- | --- | --- |
|  | InventoryControl | Return a InventoryControlWrapper to interact with the current Tab inventory. Please Note: for special Tabs, like the currency, Fragment, Map etc etc, you should use the specific class. |
|  | IsOpened | Return true if the panel is open. |
|  | StashTabInfo | Return a StashTabInfo class with info about the current Tab. |
|  | TabControl | Return a TabControlWrapper with information About the Stash Tabs. |
