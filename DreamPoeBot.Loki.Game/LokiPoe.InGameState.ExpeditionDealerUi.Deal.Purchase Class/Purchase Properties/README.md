The LokiPoe InGameState ExpeditionDealerUi Deal Purchase type exposes the following members.


|  | Name | Description |
| --- | --- | --- |

|  | IsOpen | Return true if the Purchaise window is visible |
|  | ItemYouPayName | Return a string with the name of the currency you pay. |
|  | Price | Return a int with the quantity you pay for this upgrade. |
