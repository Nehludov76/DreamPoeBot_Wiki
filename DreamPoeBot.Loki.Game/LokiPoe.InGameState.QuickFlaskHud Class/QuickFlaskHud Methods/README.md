# QuickFlaskHud Methods

The LokiPoe InGameState QuickFlaskHud type exposes the following members.


| Name | Description |
| --- | --- |
| IsFlaskInCooldown | Return the cooldown state of a specific flask slot. Pls note: slots are numbered from 1 to 5. |
| UseFlaskInSlot | Click the button for the selected flask to use it. |
