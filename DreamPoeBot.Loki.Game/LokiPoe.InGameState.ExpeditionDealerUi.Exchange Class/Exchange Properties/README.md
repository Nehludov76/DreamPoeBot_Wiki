# Exchange Properties

The LokiPoe InGameState ExpeditionDealerUi Exchange type exposes the following members.


| Name | Description |
| --- | --- |
| BuyAmmount | This member has 2 functionality, if you retrieve its value, it return the ammount of current quantity, but if you set its value, it actually change the ammount of current quantity. |
| CanAffordItem | Return true if you can buy this item (price string is not red) |
| IsOpened | Return true if the small Exchange interface is open. |
| ItemYouPayName | Return a string with the name of the currency you pay. |
| PricePerUnit | Return a double with the quantity you pay for each Unit. |
| TotalPrice | Return a double with the total quantity you pay. |
