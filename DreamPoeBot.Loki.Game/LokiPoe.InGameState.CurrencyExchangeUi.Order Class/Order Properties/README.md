# Order Properties

The LokiPoe InGameState CurrencyExchangeUi Order type exposes the following members.


|  | Name | Description |
| --- | --- | --- |
|  | BuyingName | Return the name of the currency we buy. |
|  | BuyingQuantity | Return the bought quantity. |
|  | CheapestCompetingOrder | return the current cheapperst competing order. |
|  | MyOrderRatio | return the current order ratio. |
|  | OrderStatus | The status of the order, Listed, Completed, Cancelled. |
|  | SellingName | Return the name of the currency we sell. |
|  | SellingQuantity | Retrieve the remaining quantity to sell. |
