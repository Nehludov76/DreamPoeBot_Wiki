# MercenaryEncounterUi Properties

The LokiPoe InGameState MercenaryEncounterUi type exposes the following members.


|  | Name | Description |
| --- | --- | --- |
|  | AllInventoryControls | Return a IEnumerable of InventoryControlWrapper listing ALL mercenary inventory (the main inventory and all equipped slots) |
|  | Attributes | The mercenary Attributes. |
|  | HireGoldCost | The ammount of gold to hire this mercenary. |
|  | InventoryControl_Belt | Return a InventoryControlWrapper describing your Belt inventory. This is a single slot, use CustomTabItem to get the item. |
|  | InventoryControl_Boots | Return a InventoryControlWrapper describing your Boots inventory. This is a single slot, use CustomTabItem to get the item. |
|  | InventoryControl_Chest | Return a InventoryControlWrapper describing your Chest inventory. This is a single slot, use CustomTabItem to get the item. |
|  | InventoryControl_Gloves | Return a InventoryControlWrapper describing your Gloves inventory. This is a single slot, use CustomTabItem to get the item. |
|  | InventoryControl_Head | Return a InventoryControlWrapper describing your Head inventory. This is a single slot, use CustomTabItem to get the item. |
|  | InventoryControl_LeftRing | Return a InventoryControlWrapper describing your LeftRing inventory. This is a single slot, use CustomTabItem to get the item. |
|  | InventoryControl_Main | Return a InventoryControlWrapper describing your main inventory. |
|  | InventoryControl_Neck | Return a InventoryControlWrapper describing your Neck inventory. This is a single slot, use CustomTabItem to get the item. |
|  | InventoryControl_PrimaryMainHand | Return a InventoryControlWrapper describing your PrimaryMainHand inventory. This is a single slot, use CustomTabItem to get the item. |
|  | InventoryControl_PrimaryOffHand | Return a InventoryControlWrapper describing your PrimaryOffHand inventory. This is a single slot, use CustomTabItem to get the item. |
|  | InventoryControl_RightRing | Return a InventoryControlWrapper describing your RightRing inventory. This is a single slot, use CustomTabItem to get the item. |
|  | IsOpened | Return true if the Mercenary encounter panel is open. PLS NOTE: this is not the Companion mercenary panel. |
|  | IsTakeItemActive | Return true if Take Item is enabled. |
|  | Level | The mercenary Level. |
|  | Name | The mercenary Name. |
|  | Skills | Return a list of MercenarySkill containing informations about the Mercenary skills. |
|  | Specialization | The mercenary Specialization. |
