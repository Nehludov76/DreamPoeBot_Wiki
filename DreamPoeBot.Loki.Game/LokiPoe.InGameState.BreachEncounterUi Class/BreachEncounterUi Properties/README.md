# BreachEncounterUi Properties

The LokiPoe InGameState BreachEncounterUi type exposes the following members.


|  | Name | Description |
| --- | --- | --- |
|  | BreachComleteText | The text describig the last event end result. Pls note, this might contain old data, as it is updated only after the event end, and persist till next event end. |
|  | BreachComleteTextVisible | This text is shower for few seconds at the end of the event. Check the BreachComleteText for dettails on the result. |
|  | ChargeLevelPct | Pct of charge for this event, this increase killing monsters. |
|  | DefenceLevelPct | Pct of Life in defence event, once this reach 0, the event is over. |
|  | ExaustionLevelPct | Pct of Exaustion, once this level reach the Charge level, the event end. |
|  | IsOpened | Return true if any Breach Encounter Ui (progress bar or Health bar) is displayed. |
|  | Waves | The text in the Wave text field. |
