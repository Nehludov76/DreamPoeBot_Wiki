# AtlasSkillsUi Methods

The LokiPoe InGameState AtlasSkillsUi type exposes the following members.


| Name | Description |
| --- | --- |
| CanBeAllocate | Return true if the Atlas passive can be allocated. |
| ChoosePassive | This function let you allocate a Atlas passive point. Please Note: This is the only function that has some prerequisite to work, The game window must be 800x600 and cant be Iconized, it still can be in background (hidden under other windows, not focused). |
| ConfirmOperation | Click the Confirmation button, usually used after you assign passive points. |
| IsAllocated | Return true if the Atlas passive is already allocated. |
