# MapUi Properties

The LokiPoe InGameState MapUi type exposes the following members.


| Name | Description |
| --- | --- |
| IsLargeMapVisible | Return true if the Large minimap is visible. |
| IsMiniMapVisible | Return true if the Minimap is visible. |
| LargeMapElement | Retrieve the Element informations about the Large Minimap Ui. |
| LargeMapZoom | The zoom level of the Large Minimap. |
| MiniMapElement | Retrieve the Element informations about the Minimap Ui. |
| MiniMapZoom | The zoom level of the Minimap. |
