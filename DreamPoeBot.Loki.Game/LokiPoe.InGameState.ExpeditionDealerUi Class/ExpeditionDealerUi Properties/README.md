# ExpeditionDealerUi Properties

The LokiPoe InGameState ExpeditionDealerUi type exposes the following members.


|  | Name | Description |
| --- | --- | --- |
|  | AvailableExpeditionCurrency | Return a list of ExpeditionAvailableCurrency that describe the name and quantity of currency available for this specific Dealer. |
|  | CanRefreshItems | Return true if the Refresh button is enabled. |
|  | InventoryControl | Return a InventoryControl that let you interact with the current Tab content. |
|  | IsOpened | Return true if the panel is open. |
|  | TabControl | Return a TabControlWrapper that describe the Tab controll of the panel. |
|  | Title | Return the Main Window Title (Haggle, Gamble, Deal, Exchange) |
