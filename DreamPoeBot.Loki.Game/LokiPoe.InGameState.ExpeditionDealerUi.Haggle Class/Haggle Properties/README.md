# Haggle Properties

The LokiPoe InGameState ExpeditionDealerUi Haggle type exposes the following members.


|  | Name | Description |
| --- | --- | --- |
|  | BetAmmount | This member has 2 functionality, if you retrieve its value, it return the ammount of current bet, but if you set its value, it actually change the ammount of current bet. |
|  | CanAffordItem | Return true if you can buy this item (price string is not red) |
|  | IsOpened | Return true if the small Hangle interface is open. |
|  | ItemYoubuy | Return a Item class that contain info about the item you buy. |
|  | ItemYouPayName | Return a string with the name of the currency you pay. |
|  | MaxBet | Return the max quantity the you can bet. Please note, after the first bet this number change and is usually decreased (the game show a red vertical bar over the slider) |
|  | MinBet | Return the min quantity the you can bet. Please note, after the first bet this number change and is usually increased (the game show a red vertical bar over the slider) |
