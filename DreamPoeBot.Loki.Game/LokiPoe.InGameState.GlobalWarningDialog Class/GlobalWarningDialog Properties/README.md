# GlobalWarningDialog Properties

The LokiPoe InGameState GlobalWarningDialog type exposes the following members.


|  | Name | Description |
| --- | --- | --- |
|  | ActionText | Return the text of the accept button. |
|  | BodyText | Return the body of the message. |
|  | CancelText | Return the text of the cancell button. |
|  | IsAbandonLabyrintWarningOverlayOpen | Return true if you try to leave a labyrinth. |
|  | IsBetrayalLeaveZoneWarningOverlayOpen | Return true if you try to leave the Immortal Syndicate Safehouse. |
|  | IsDestroyItemWarningOverlayOpen | Return true if you try to restroy a item. |
|  | IsDismissMasterWarningOverlayOpen | Return true if you try to leave a HO that has an active master mission (usually the Temple portals, or the Syndacate safehouse portals). |
|  | IsOpened | Return true if a warning message is displayed. |
|  | IsPassiveTreeResetWarningOverlayOpen | Return true if the message is a Passive Tree Reset Warning. |
|  | IsPassiveTreeWarningOverlayOpen | Return true if you try to reset your passive tree. |
|  | IsPassiveWarningOverlayOpen | Return true if your passive tree is been resetted (usually after a relevant game update). |
|  | IsSealProphecyOverlayOpen | Return true if you try to seal a prophecy. |
|  | IsTradingItemWithGemsOverlayOpen | Return true if you try to trade a item sith a socketed gem. |
|  | IsVendoringItemWithGemsOverlayOpen | Return true if you try to sell a item with a socketed gem. |
|  | IsVendoringItemWithIncubatorsOverlayOpen | Return true if you try to sell a item with a incubator socketed. |
|  | TitleText | Return the title of the message. |
