# Atlas Properties

The LokiPoe InstanceInfo Atlas type exposes the following members.


|  | Name | Description |
| --- | --- | --- |
|  | AllAreas | Retrieve a List containing all the Maps in the atlas as DatWorldAreaWrapper |
|  | AlvaNormalMissionsLeft | Retrieve the number of white Alva missions left. |
|  | AlvaRedMissionsLeft | Retrieve the number of red Alva missions left. |
|  | AlvaYellowMissionsLeft | Retrieve the number of yellow Alva missions left. |
|  | AtlasInfluences | Retrieve a list of AtlasInfluence with info about different influences on the atlas. |
|  | AwakeningBonusCompletedAreas | Retrieve a list of all Maps in the atlas where you successfully completed the Awakening bonus objective. |
|  | BonusCompletedAreas | Retrieve a list of all Maps in the atlas where you successfully completed the bonus objective as AtlasNodeRecord, Normal and Unique zones. |
|  | BonusCompletedNormalAreas | Obsolete. Retrieve a list of all normal Maps in the atlas where you successfully completed the bonus objective as DatWorldAreaWrapper, Normal zones only. |
|  | CompletedAreas | Obsolete. This member is a copy of AreasWhereMavenHoldARecreation, it is here for back compatibility but should not be used. The reason for it existance and name is bury in the history of the game, and how radically the Atlas has changed in Poe. |
|  | EinharNormalMissionsLeft | Retrieve the number of white Einhar missions left. |
|  | EinharRedMissionsLeft | Retrieve the number of red Einhar missions left. |
|  | EinharYellowMissionsLeft | Retrieve the number of yellow Einhar missions left. |
|  | ElderInfluencedAreas | Obsolete. |
|  | ExploredAtlasNodes | Return a list of AtlasNodeRecord representing all nodes that the player as opened at least once. |
|  | JunNormalMissionsLeft | Retrieve the number of white Jun missions left. |
|  | JunRedMissionsLeft | Retrieve the number of red Jun missions left. |
|  | JunYellowMissionsLeft | Retrieve the number of yellow Jun missions left. |
|  | KiracNormalMissionsLeft | Retrieve the number of white Kirac missions left. |
|  | KiracRedMissionsLeft | Retrieve the number of red Kirac missions left. |
|  | KiracYellowMissionsLeft | Retrieve the number of yellow Kirac missions left. |
|  | MavenSubmapCompleted | Retrieve a List of all Maven sub map boss completed |
|  | MemoryLines |  |
|  | NikoNormalMissionsLeft | Retrieve the number of white Niko missions left. |
|  | NikoRedMissionsLeft | Retrieve the number of red Niko missions left. |
|  | NikoYellowMissionsLeft | Retrieve the number of yellow Niko missions left. |
|  | NrOfSocketedVoidstones | Return the number of Voidestones socketed in the Atlas. |
|  | ShapedAreas | Obsolete. |
|  | ShaperInfluencedAreas | Obsolete. |
|  | SocketedAtlasVoidstones | Return a List of InGameState.AtlasUi.AtlasVoidstone with info about the Name of the Voidstone and Region influenced. |
|  | UncompletedAreas | Retrieve a List of all uncompleted Maps in the atlas as DatWorldAreaWrapper |
|  | UncompletedAwakeningBonusAreas | Retrieve a list of all Maps in the atlas where you don't have completed the Awakening bonus objective. |
|  | UncompletedBonusAreas | Retrieve a list of all Maps in the atlas where you don't have completed the bonus objective. |
|  | ZanaNormalMissionsLeft | Obsolete. Retrieve the number of white Zana missions left. |
|  | ZanaRedMissionsLeft | Obsolete. Retrieve the number of red Zana missions left. |
|  | ZanaYellowMissionsLeft | Obsolete. Retrieve the number of yellow Zana missions left. |
