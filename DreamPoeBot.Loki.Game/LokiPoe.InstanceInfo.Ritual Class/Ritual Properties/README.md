# Ritual Properties

The LokiPoe InstanceInfo Ritual type exposes the following members.


|  | Name | Description |
| --- | --- | --- |
|  | IsRitualActive | Return true if the Ritual is active |
|  | RitualsRemaining | Return the number of rituals remaining in the current zone. Pls Note: It always return 3 if you have not activate any, as soon as you complete 1 ritual, it start to give you exact numbers. There is no way to know if a zone contain rituals before you encounter one. |
|  | Tributes | Return the number of actually collected tributes |
