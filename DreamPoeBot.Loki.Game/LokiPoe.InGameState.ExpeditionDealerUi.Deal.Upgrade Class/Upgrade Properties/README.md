# Upgrade Properties

The LokiPoe InGameState ExpeditionDealerUi Deal Upgrade type exposes the following members.


| Name | Description |
| --- | --- |
| CanAffordUpgrade | Return true if you can buy this item (price string is not red) |
| IsOpen | Return true if the Item Upgrade window is visible |
| IsSkipButtonEnabled | Return true if the Skip button is enabled. |
| IsTakeItemButtonEnabled | Return true if the TakeItem button is enabled. |
| IsUpgradeButtonEnabled | Return true if the Upgrade button is enabled. |
| ItemDescription | Return the full description of the item, like displayed in th item tooltip. |
| ItemYouPayName | Return a string with the name of the currency you pay. |
| Price | Return a int with the quantity you pay for this upgrade. |
